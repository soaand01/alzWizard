Phase 1 — Tools & Connectivity

Install the required CLI tools and confirm internet access before starting bootstrap.

Purpose: Ensure your workstation or CI environment has the tools and network access required to run the accelerator.

Before you start
  • Workstation or build agent with administrator privileges to install software.
  • Open outbound internet access to required Azure endpoints or a configured proxy.
Required tools
  1. PowerShell 7.4 or later — install: Install PowerShell.
  2. Azure CLI 2.55.0 or later — install: Install Azure CLI.
  3. Git CLI — install: Git Downloads.
Verify
  • From PowerShell Core (pwsh) confirm the tools are in PATH and respond: 
    pwsh -c "az --version && git --version && pwsh --version"
  • Confirm outbound connectivity to Azure endpoints from your environment.
