Get Started

Quick checklist to plan and prepare for the Azure Landing Zones Accelerator.

Purpose: This companion guide condenses the Accelerator user guide into a concise workflow: Plan → Prepare → Bootstrap → Run. Use this page to orient your team and navigate to the appropriate phase.

Before you start
  • Administrator access: Ensure you or a teammate has tenant administrator permissions to create app registrations and assign roles.
  • Decide IaC: Choose Terraform or Bicep for your deployment and record that choice.
  • Version control: Select a VCS provider (Azure DevOps or GitHub) and confirm organization-level access if required.
Steps
  1. Open the official ALZ Accelerator user guide: Azure Landing Zones Accelerator — User Guide.
  2. Download and complete the planning checklist (checklist.xlsx). Record management groups, subscription names, regions, and the starter module.
  3. Proceed to Phase 0 for planning guidance or go directly to Phase 1 to prepare tools and permissions.
Verify
  • Planning checklist saved and starter choices recorded.
  • Administrator contact identified for app registrations and role assignments.

Phase 0 is optional but recommended for production deployments. Use the links above for detailed guidance.
